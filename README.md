# 🚗 Miniguia de Estudos para CNH (Modelo Autônomo 2026)

Este repositório foi desenvolvido para o desafio da **DIO**, utilizando o **NotebookLM** para auxiliar candidatos à CNH no novo modelo de estudo independente.

## 🎯 Contexto e Objetivos
Com o fim da obrigatoriedade das autoescolas para exames teóricos em 2026, este projeto organiza o conhecimento do CTB em um guia prático e interativo via IA.

## 📚 Curadoria de Fontes
* Código de Trânsito Brasileiro (CTB) 2026.
* Manuais de Direção Defensiva e Primeiros Socorros do CONTRAN.

## 🧠 Engenharia de Prompts e "Cicatrizes"
* **Prompt Principal:** "Atue como instrutor e resuma regras de preferência."
* **Ajustes:** Foi necessário restringir a busca para a legislação brasileira de 2026.
